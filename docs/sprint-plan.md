## 1. 范围裁剪（必看）

**5 天交付必须做减法**，下面这些**全部砍掉**：

| ❌ 砍掉 | 替代方案 |
|---|---|
| pen/coin 自定义训练 | 仅用 COCO 12 类，pen/coin **不支持** |
| Room 本地缓存 | 纯网络调用，进入页面时刷新 |
| 引导页 | 跳过，App 直接进首页 |
| 缩略图优化 | 列表用原图（已在后端缩放至 1024px） |
| Dockerfile / docker-compose | 本地跑通即可，附 `start.sh` 即可 |
| 性能压测 / locust | 手动 3 并发不卡就算通过 |
| ProGuard / 签名 | Debug APK 可装可跑 |
| 大量单元测试 | 各 1~2 个 smoke test |
| Swagger 文档整理 | 用 FastAPI 默认 UI |
| 多语言 / 深色模式 | 不做 |
| 图片清理任务 | MVP 不实现 |
| 性能优化周（W6） | 全部砍 |

**保留（核心 MVP）：**
- ✅ 拍照 / 相册选图 / EXIF 校正
- ✅ 真模型推理
- ✅ 框选显示 + 类别 + 计数
- ✅ 手动 +/- / 长按删除
- ✅ 历史记录（简单列表）
- ✅ 端到端跑通
- ✅ Demo 视频 + 答辩 PPT
